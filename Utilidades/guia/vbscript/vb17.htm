Controles HTML intrínsecos

Bajo esta denominación se encuentran los elementos típicos de los formularios, que ya conocemos. Precisamente por ser elementos intrínsecos de HTML carecen de propiedades o métodos, es decir, no son objetos, son elementos contenidos dentro del objeto Form, y ya hemos visto cómo se puede controlar sus atributos NAME y VALUE desde las propiedades de este objeto. Sí disponen de algunos eventos:

Este es el significado de los eventos:

  • OnClick Se produce cuando se pulsa con el botón izquierdo del ratón sobre el elemento.
  • OnFocus Se produce cuando el elemento recibe el enfoque, es decir, cuando el cursor está situado sobre él, pero no cuando el ratón está sobre él. Por tanto, solamente se produce después de haber pulsado el ratón.
  • OnBlur Es lo contrario del anterior, es decir, cuando el elemento deja de tener el enfoque, por ejemplo, después de pulsar la tecla de tabulación.
  • OnChange Se produce cuando el valor actual del control cambia de cualquier forma: de vacío a valor o de valor a otro valor diferente o a vacío
  • OnSelect Cuando el contenido del elemento es seleccionado, es decir cuando cambia el color del fondo. No es lo mismo que recibir el enfoque, aunque necesariamente OnFocus se producirá antes.
La sintaxis de todos estos eventos es la misma que ya se ha visto en páginas y ejemplos anteriores.
