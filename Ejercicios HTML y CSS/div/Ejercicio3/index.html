Ejemplo del uso de los elementos Div & span Ejemplo 3 de <div>

La etiqueta <div>

Usamos la etiqueta <div> para agrupar dos párrafos y aplicar color de fondo (background) al texto,
y para añadir color a esta palabra la colocamos dentro de la etiqueta <span>.

Presta atención, que la etiqueta <div> es un elemento de nivel de bloque,
por lo que tiene un salto de línea antes y después.
